# Globex Website Redesign

## Objective

Redesign the Globex storefront to lift trade-buyer conversion and make the catalog easy to browse on mobile.

## Context

Globex sells spare parts and equipment to professional buyers. The current site is desktop-first, slow on mobile, and its search rarely surfaces the right part. This project rebuilds the front end and the product discovery flow while keeping the existing order backend.

## Planned activities

- [ ] Discovery and content audit of the current catalog
- [ ] Wireframes for the home, category, and product pages
- [ ] Visual design and component library
- [ ] Front-end build and catalog search integration
- [ ] User testing with a small panel of trade buyers
- [ ] Launch and post-launch monitoring
