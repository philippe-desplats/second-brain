# Client Onboarding

## Observations

- [definition] The sequence that takes a new client from signed contract to a productive working relationship, usually the first one to two weeks
- [example] The Globex website redesign onboarding: scope confirmation, asset collection, kickoff, cadence setup, recap
- [opinion] The first two weeks set the tone, a tight kickoff and a written recap prevent most early friction
